# Changelog

## [1.0.0] - 2026-01-15

Initial release.

### Features

- **Smart Analysis** — Pre-interview file analysis with CLEAR/UNCLEAR/MISSING classification
- **8-Stage Interview** — Problem, Users, Functional, UI/UX, Edge Cases, Non-Functional, Technical, Prioritization
- **Validation Checklist** — 14-category best practices check before writing spec
- **Complexity Analysis** — Score-based complexity detection with split recommendations
- **Context Scanning** — Auto-detects tech stack from project files
- **Multi-Language** — Auto-detection for 12+ languages
- **Adaptive Depth** — Adjusts explanations based on technical level
- **14-Section PRD Output** — Comprehensive specification document template
